# 自主能力阶梯

英文版见 [capability-ladder.en.md](capability-ladder.en.md)。

## L0 基础验证

- 读健康状态
- 读页面
- 读公共 API
- 读 artifacts

## L1 受控动作

- 登录管理员页面
- 触发 `pause`
- 触发 `resume`
- 触发 `run-now`
- 触发 `flatten`

## L2 证据录制

- 录制失败视频
- 录制成功视频
- 导出截图
- 导出 trace
- 导出 JSON 报告

## L3 真实链路验证

- 真实服务存活
- 真实 run 持久化
- 真实 public site 回显
- 真实 portfolio sync
- 真实小额 smoke trade

## L4 自主开发闭环

- 自动验证仓库状态
- 自动复现缺陷
- 自动提出修复候选
- 自动再验证
- 自动响应反馈
- 自动修复构建失败

## L5 人工仅在判断点介入

- 只有策略判断、风险授权、合并审批才交人工
- 其他步骤由系统自动推进
