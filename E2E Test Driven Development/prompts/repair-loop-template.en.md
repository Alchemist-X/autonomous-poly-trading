# Autonomous Repair Loop Prompt Template

Chinese version: [repair-loop-template.md](repair-loop-template.md)

This template is reserved for phase two. It is intended to standardize the prompt input for the full loop of detect -> reproduce -> fix -> verify.

Suggested fields:

- target issue
- impact scope
- known reproduction steps
- required failure evidence
- success criteria
- allowed change scope
- prohibited dangerous operations
