# Autonomy Capability Ladder

Chinese version: [capability-ladder.md](capability-ladder.md)

## L0 Basic verification

- read health state
- read pages
- read public APIs
- read artifacts

## L1 Controlled actions

- log into the admin page
- trigger `pause`
- trigger `resume`
- trigger `run-now`
- trigger `flatten`

## L2 Evidence recording

- record failure videos
- record success videos
- export screenshots
- export traces
- export JSON reports

## L3 Real-path validation

- real service health
- real run persistence
- real public site reflection
- real portfolio sync
- real low-notional smoke trade

## L4 Autonomous development loop

- automatically validate repository state
- automatically reproduce defects
- automatically propose fixes
- automatically verify fixes
- automatically respond to feedback
- automatically repair build failures

## L5 Human intervention only at judgment points

- strategy judgment, risk authorization, and merge approval remain human steps
- everything else moves forward automatically
